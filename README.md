# Daily Envelope Budget

Local-only PWA: monthly budget split across days in the calendar month, with rollover within the month (not across months). Amounts are shown in **CAD**. Includes a 7-day wishlist cooldown and JSON backup import/export.

## Develop

```bash
npm install
npm run dev
```

## Test

```bash
npm run test
```

## Build

```bash
npm run build
npm run preview
```

Data stays in IndexedDB in your browser. Use **Settings → Export JSON** for backups.
